DeepVid!

The project builds an efficient code for action recognition (humans) using UCF101 (http://crcv.ucf.edu/data/UCF101.php) with the help of Keras(https://keras.io/). Convolution Neural Networks are used for training the dataset. Also, accuracies are compared for different pre-trained models on [Imagenet](http://www.image-net.org/). Videos can be viewed as a sequences of images(frames).
